Install Guide

1. Install Python, Homebrew, PyQt5, Numpy, and OpenCV
	Follow the instructions with this link: 
	https://www.learnopencv.com/install-opencv-3-4-4-on-macos/
2. Clone GitHub branch
    1. Git clone https://github.com/AguaClara/floc_app.git
3. Run 'camera.py' in the folder 'new_camera' and if everything runs then you are good to go!
	If PyQt5 does not work, try:
	1. brew uninstall qt
	   brew install PyQt5
	OR
	2. pip3 install PyQt5
